Package for simulating X-ray Difraction from polycrystals¶

  • Defines polycrystals as a mesh of tetrahedral single crystals.

  • Simulates diffraction based on structure factors and a discretized detector array.

xrd_simulator¶

The beam class defines an X-ray as a 3d spatial (convex) object by taking a set of points and intepreting them as the convex hull of the beam cross section. The beam is then extruded in a given direction towards infinity.

The detector class takes a set of scattere-rays objects and produces a discrete pixelated framestack.

A mesh object defines a tetrahedral set of 3d elements and tracks their respective normals and nodes.

find_solutions_to_tangens_half_angle_equation(k1, k2, c_0, c_1, c_2)¶

Find all solutions, s, to the equation

\[(c_2 - c_0) t^2 + 2 c_1 t + (c_0 + c_2) = 0. \quad\quad (1)\]

where

\[t = \tan(s \alpha / 2). \quad\quad (2)\]

and .. math::alpha is the angle between k1 and k2

Parameters

  • k1 (numpy array) – Intervall start of illuminating wave vectors. (shape=(3,))

  • k2 (numpy array) – Intervall end of illuminating wave vectors. (shape=(3,))

  • c_0,c_1,c_2 (float) – Equation coefficents

Returns

solutions (s1, s2) if any exists otehrwise returns (None, None).

Return type

(float or None)

get_kprimes(k1, k2, U, B, G_hkl)¶

Check for difraction from a crystal for a specific hkl family and illumination intervall, [k1,k2].

The intervall is defined by rotating the illuminating wave vector k1 into k2 in the plane spanned by k1 and k2. The crystal is defined by its U and B matrix.

Parameters

  • k1 (numpy array) – Intervall start of illuminating wave vectors. (shape=(3,))

  • k2 (numpy array) – Intervall end of illuminating wave vectors. (shape=(3,))

  • U (numpy array) – Crystal coordinates to sample coordinates mapping matrix. (shape=(3,3))

  • B (numpy array) – Crystal reciprocal coordinates to real coordinates mapping matrix. (shape=(3,3))

  • G_hkl (numpy array) – Diffraction (or scattering) Miller indices. (shape=(3,))

Returns

Solutions to diffraction equations for the given crystal and the given parametric intervbal of illumination. If no solutions exist None is returned.

Return type

(numpy array or None)

get_parametric_k(k1, k2, s)¶

Finds vector k by rotating k1 towards k2 a fraction s in the plane spanned by k1 and k2.

\[\boldsymbol{k}(s) = \boldsymbol{k}_i\cos(s\alpha)+(\boldsymbol{\hat{r}}\times \boldsymbol{k}_i)\sin(s\alpha)\]
Parameters

  • k1 (numpy array) – Intervall start of illuminating wave vectors. (shape=(3,))

  • k2 (numpy array) – Intervall end of illuminating wave vectors. (shape=(3,))

  • s (float) – Angular rotation fraction (s=0 gives k1 and s=1 gives k2 as result).

Returns

k, the wave vector in the intervall of k1 and k2.

Return type

(numpy array)

get_tangens_half_angle_equation(k1, k2, theta, G)¶

Find the coefficents of the quadratic equation

\[(c_2 - c_0) t^2 + 2 c_1 t + (c_0 + c_2) = 0. \quad\quad (1)\]

Its roots (if existing) are solutions to the Laue equations where t is

\[t = \tan(s \alpha / 2). \quad\quad (2)\]

and s is the sought parameter which parametrises all wave vectors between k1 and k2 where s=0 corresponds to k1 and s=1 to k2. Intermediate vectors are reached by rotating k1 in the plane spanned by k1 and k2.

Parameters

  • k1 (numpy array) – Intervall start of illuminating wave vectors. (shape=(3,))

  • k2 (numpy array) – Intervall end of illuminating wave vectors. (shape=(3,))

  • theta (float) – Bragg angle in radians of sought diffraction.

  • G (numpy array) – Diffraction (or scattering) vector of sought diffraction. (shape=(3,))

Returns

Coefficents c_0,c_1 and c_2 of equation (1).

Return type

(tuple of float)

The scatterd ray object is not a simmlified object but any approximations are meant to be made during detector rendering.
